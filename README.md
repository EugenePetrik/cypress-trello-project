# Cypress advanced project

### Requirements
- NodeJS
- GIT

### Install project dependencies

```bash
npm install
```

### Run application server

```bash
npm run start
```

### Run tests

```bash
npm run cy:open
```

### Run tests in headless mode

```bash
npm run cy:run
```

### Generate allure report and open in browser

```bash
npm run allure:report
```
